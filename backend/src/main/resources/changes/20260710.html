Find datasheet

The part detail page's Documents card now has a Find datasheet button next to the existing upload controls. It searches the Internet for candidate datasheet PDFs — the same kind of search already used to find part photos — and shows a picker of results so you can save the right one with a single click.

Tags

Parts can now be tagged with one or more free-form labels. Tags are case-independent (typing "smd" reuses the existing "SMD" tag rather than creating a duplicate) and can be multiple words. While typing a tag, existing tags matching what you've typed so far are suggested once you've entered at least 3 characters; if you enter a tag that doesn't exist yet, you'll be asked to confirm before it's created. Tags can be added when creating or editing a part (Parts page, Quick Add, and the part detail page) and are shown as badges on the parts list and part detail page.
