Stock thresholds redesigned

Minimum stock levels are now tracked per root location rather than per individual shelf or sub-location. This means the threshold for a part in "Building A" covers all stock spread across rooms and cupboards inside it.

  • The old per-location minimums have been migrated automatically (highest value wins when multiple sub-locations had a threshold).
  • Set or edit thresholds from the Stock Thresholds card on any part's detail page.
  • The Low Stock page now shows the total on-hand quantity across the whole subtree compared to the threshold.
  • Thresholds can only be placed on root locations (those with no parent).
